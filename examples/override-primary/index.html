jds override-primary

Same JDS, in teal.

Five semantic tokens, two modes, zero patches. Cursor, primary button, focus ring, syntax 'function' flip blue → teal. For JDS's 7 built-in hues at runtime, use [data-accent] instead — see multi-accent demo.

Demonstration

Primary is now teal

All five overridden tokens cascade down: button background, card title, focus ring, code "function" color, and the cursor in the header.

The override (drop into your project's CSS)
css· override.css 
/* Override JDS semantic tokens. JDS unchanged. */
:root,
[data-theme="dark"] {
  --jds-accent-primary:    #14b8a6;
  --jds-accent-primary-bg: rgba(20, 184, 166, 0.12);
  --jds-border-focus:      #14b8a6;
  --jds-shadow-focus:      0 0 0 3px rgba(20, 184, 166, 0.35);
  --jds-syntax-function:   #14b8a6;
}
[data-theme="light"] {
  --jds-accent-primary:    #0f766e;
  --jds-accent-primary-bg: rgba(15, 118, 110, 0.10);
  --jds-border-focus:      #0f766e;
  --jds-shadow-focus:      0 0 0 3px rgba(15, 118, 110, 0.30);
  --jds-syntax-function:   #0f766e;
}
