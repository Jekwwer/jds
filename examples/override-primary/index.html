Demonstration

Primary is now red

All five overridden tokens cascade down: button background, card title, focus ring, code "function" color, and the cursor in the header.

The override (drop into your project's CSS)
css · override.css 
/* Path A: override JDS semantic tokens. JDS unchanged. */
:root,
[data-theme="dark"] {
  --jds-accent-primary:    var(--jds-red);
  --jds-accent-primary-bg: rgba(224, 108, 117, 0.12);
  --jds-border-focus:      var(--jds-red);
  --jds-shadow-focus:      0 0 0 3px rgba(224, 108, 117, 0.35);
  --jds-syntax-function:   var(--jds-red);
}
[data-theme="light"] {
  --jds-accent-primary:    var(--jds-red-light);
  --jds-accent-primary-bg: rgba(197, 48, 48, 0.10);
  --jds-border-focus:      var(--jds-red-light);
  --jds-shadow-focus:      0 0 0 3px rgba(197, 48, 48, 0.30);
  --jds-syntax-function:   var(--jds-red-light);
}
